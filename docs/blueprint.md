# **App Name**: Averon Market PY

## Core Features:

- Product Listing: Mostrar productos con imágenes, nombres, precios, descripciones y etiquetas de oferta.
- Shopping Cart: Implementar un carrito de compras persistente para guardar los productos seleccionados.
- Checkout with Google Pay: Integrar Google Pay para pagos en línea y confirmación automática de pedidos.
- Checkout with COD: Habilitar el pago contra entrega con registro de pedidos en Firestore como 'Pendiente de pago'.
- User Authentication: Permitir a los usuarios registrarse, iniciar sesión, ver el historial de pedidos y gestionar los datos de contacto.
- Product Search and Filters: Permitir a los usuarios buscar y filtrar productos por categoría, precio y disponibilidad.
- Offer Notifications: Herramienta de IA que notifica condicionalmente a los usuarios sobre nuevas ofertas.

## Style Guidelines:

- Color primario: Azul profundo (#1F3A93) para transmitir confianza y profesionalismo.
- Color secundario: Naranja brillante (#FFA500) para resaltar botones y ofertas especiales.
- Color de fondo: Blanco (#FFFFFF) para una apariencia limpia y moderna.
- Fuente: 'Inter', una sans-serif (titular y cuerpo) con un diseño limpio, moderno y legible para garantizar confianza y profesionalismo.
- Usar iconos modernos y claros para mejorar la navegación y guiar las acciones del usuario.
- Mantener un diseño limpio y minimalista con un menú superior fijo y un pie de página completo. Asegurar un diseño totalmente responsivo para todos los dispositivos.
- Usar animaciones sutiles en los hovers de los botones y las transiciones para mejorar la experiencia del usuario sin sobrecargar la interfaz.
